Coloque aqui os ficheiros .traineddata do Tesseract OCR.

Exemplo:
  por.traineddata  → Português
  eng.traineddata  → Inglês
  lat.traineddata  → Latim

Download (versão fast, ~5MB por idioma):
  https://github.com/tesseract-ocr/tessdata_fast

Download (versão completa, ~30MB por idioma, mais precisa):
  https://github.com/tesseract-ocr/tessdata

Os ficheiros serão copiados automaticamente para uma pasta temporária
em runtime e usados pelo motor OCR sem necessidade de instalar o Tesseract.
